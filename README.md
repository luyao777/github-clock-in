# github-clock-in
无聊的Github打卡器(Boring Github clock-in)

## Description
有了它，你也能让你的贡献日历充满浓浓绿意了。

With it, you can also make your contribution calendar full of green.

また、寄付カレンダーを緑でいっぱいにすることもできます。

## Usage
`python clock-in.py`
